# Ek Bharat Shreshtha Bharat (EBSB) Club Website - IIT Jammu

Welcome to the official repository for the EBSB Club website at IIT Jammu. This project aims to showcase the club's activities, events, and cultural initiatives under the Ek Bharat Shreshtha Bharat program.

## Overview

The website is designed with a modern, clean, and culturally resonant aesthetic, using a color palette inspired by Saffron, White, Green, and Navy. It serves as a central hub for all club-related information.

## Project Structure

The website consists of the following main sections:
- **Home**: Landing page with an overview of the club.
- **About Us**: Information about the club's mission and vision.
- **Team**: Profiles of the club's core members.
- **Events**: Details on upcoming and past cultural events.
- **Gallery**: A visual showcase of club activities.
- **Blog**: Articles and updates related to EBSB initiatives.
- **Resources**: Helpful materials and links.
- **Achievements**: Highlighting the club's milestones.
- **Contact**: Get in touch with the team.

## Technologies Used

- **HTML5**: For semantic page structure.
- **CSS3**: For modern styling, animations, and responsive layout.
- **JavaScript**: For interactive elements like mobile navigation, smooth scrolling, and dynamic content filtering.

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Princu-Babu/ebsb-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ebsb-website
   ```
3. Open `index.html` in your preferred web browser. No local server is strictly required, though a tool like Live Server (VS Code extension) is recommended for development.
